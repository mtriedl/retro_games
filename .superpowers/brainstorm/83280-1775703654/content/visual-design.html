Key Visual Elements

  • EGA color palette -- buildings use the classic 16-color EGA palette (bright red, blue, green, cyan, magenta, brown)
  • Lit/unlit windows -- randomly toggled yellow/dark, just like the original
  • Banana emoji with rotation animation during flight (modern touch)
  • Faint shot trail (toggleable) -- dots that fade from the launch point
  • Off-screen tracker -- yellow arrow at top follows banana's X when it's above viewport
  • Explosion crater -- circular cutout in buildings where hits land
  • Wind arrows -- visual indicator scaling with strength, replacing the original's plain number
  • Input panel -- bottom-left overlay with prefilled values and last-turn reference
